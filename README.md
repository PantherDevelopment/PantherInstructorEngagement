# Panther Instructor Engagement Reports

A desktop application for Canvas LMS administrators to monitor instructor activity and engagement across online courses. Built for Florida Institute of Technology.

## Download

Download the latest installer from the [Releases](https://github.com/PantherDevelopment/PantherInstructorEngagement/releases/latest) page:

- **Mac** — download `PantherInstructorEngagement-x.x.x-macOS.dmg`, open it and drag the app to your Applications folder
- **Windows** — download `PantherInstructorEngagement-x.x.x-windows-setup.exe` and run the installer

## First-Time Setup

On first launch you will be prompted for:
1. Your institution's Canvas URL (e.g. `https://fit.instructure.com`)
2. A Canvas API token — generate one in Canvas under **Account → Settings → New Access Token**

Your credentials are stored securely in your system keychain.

## Usage

1. Enter at least one filter (Course Code, Instructor, Year, Semester, or Term)
2. Click **Search** to find matching courses
3. Click courses in the results list to select them (click again to deselect)
4. Choose **Current Week** or **Entire Term** report period
5. Click **Generate Engagement Report** and choose where to save the Excel file

See **About → User Guide** in the application for full documentation.

## Updates

The app checks for updates automatically on launch. You can also check manually via **About → Check for Updates**.
